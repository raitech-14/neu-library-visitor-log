<h1>📚 NEU Library Visitor Log </h1>

<p>
A web-based Visitor Log designed for New Era University Library.
This system allows students, faculty, staff, and visitors to check in digitally and record their visit details.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
  <li>🔐 Visitor check-in system</li>
  <li>📧 Institutional email validation (@neu.edu.ph only)</li>
  <li>👤 Automatic user registration and update</li>
  <li>🚫 Blocked user detection</li>
  <li>📝 Visit logging (date & time)</li>
  <li>🎯 Dynamic purpose selection based on visitor type</li>
  <li>🖥️ Clean and responsive UI</li>
</ul>
